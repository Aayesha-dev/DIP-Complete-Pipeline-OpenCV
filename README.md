# Digital Image Processing - Complete Pipeline Project

## 📌 Overview
This project is a **complete Digital Image Processing (DIP) pipeline** that demonstrates various image processing techniques using Python and OpenCV. It covers the entire workflow from basic preprocessing to advanced object detection.

---

## 🚀 Features

| Feature | Description |
|---------|-------------|
| ✅ **Grayscale Conversion** | Convert RGB images to grayscale |
| ✅ **Binary Image Conversion** | Convert grayscale images to binary (thresholding) |
| ✅ **Noise Addition** | Add Gaussian and Salt & Pepper noise |
| ✅ **Noise Removal** | Remove noise using Median and Gaussian Blur filters |
| ✅ **Image Enhancement** | Histogram Equalization and Contrast Enhancement |
| ✅ **Object Detection** | Detect objects using OpenCV (Haar Cascade / Contour Detection) |
